# Ambient Sound Mixer Widget

## Setup
1. Place audio files in `assets/audio/`:
    - `wind.mp3`
    - `rain.mp3`
    - `birds.mp3`
    - `waves.mp3`
    - `fire.mp3`
2. Open `index.html` in a web browser.

## Features
- **Master Control**: Play/Pause all sounds and control master volume.
- **5-Track Mixer**: Individual volume controls for Wind, Rain, Birds, Waves, and Fire.
- **Smart Audio**: Uses Web Audio API for gapless looping and smooth 2-second crossfades.
- **Responsive Design**: Works on Desktop and fits into small Notion columns (horizontal scroll).
- **Visual Feedback**: Icons change opacity and glow intensity based on volume.
